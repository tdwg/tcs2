# Rothfels & al. 2012 Eupolyploids recipe

[Recipes](../recipes)

[rothfels_revised_2012](../bibliography/#rothfels_revised_2012)) is one of very
few botanical taxonomic treatments that include mappings to concepts in previous
treatments.
