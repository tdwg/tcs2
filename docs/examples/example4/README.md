# Example 4

## Amblystegium and related genera

From:
> **Koperski, Monica, Michael Sauer, Walter Braun, and S. Rob Gradstein (2000).** Referenzliste der Moose Deutschlands, Dokumentation unterschiedlicher taxonomischer Auffassungen. *Schriftenreihe für Vegetationskunde* **34**. Bonn-Bad Godesberg: Bundersamt für Naturschutz.

The *Reference list of Germany's bryophytes* by Koperski and others is one of 
the earliest publications that presented taxon relationships (which the 
publication called 'concept synonyms' ('Konzeptsynonyme')) at scale and is 
probably still the largest set of taxon references.

The below example featuring Amblystegium and related genera was one of the 
examples highlighted in the introduction and was partially reproduced by Franz & 
Peet (2009).
